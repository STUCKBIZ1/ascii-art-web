# ASCII Art Web

## Description

Ascii-art-web is a web application written in Go and html that allows users to generate ASCII art from text using different banner styles. The application provides a simple web interface where users can input text, select a banner style, and generate ASCII art, which is then displayed on the page.

Supported banners:
- standard
- shadow
- thinkertoy
